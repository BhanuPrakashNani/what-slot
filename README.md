## Usage
Currently hosted at [https://what-slot-kgp.herokuapp.com/](https://what-slot-kgp.herokuapp.com/)
## Runnning locally
1. `git clone https://github.com/arnav-t/what-slot.git`
2. `cd what-slot`
3. `pipenv shell --three`
4. `pipenv install` (Only the first time)
5. `python3 app.py`
